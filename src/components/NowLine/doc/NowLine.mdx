## Usage

```ts
import { NowLine } from 'remindee-ui-kit';

export const Example = () => {
  return <NowLine text="Now" />;
};
```

### Arguments

- **text** - A string value indicating text message over blue line.
- **top** - A number value which show where NowLine will be placed.
- **className** - A string that sets css class to the root of component.
