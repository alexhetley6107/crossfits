# Skeleton

<Canvas of={Skeleton.Base} />

## Usage

```ts
import { Skeleton } from 'remindee-ui-kit';

export const Example = () => {
  return <Skeleton />;
};
```

### Arguments

- **className** - A string that sets css class to the root of component.
- **width** - A number that sets width of skeleton in `px`.
- **height** - A number that sets height of skeleton in `px`.
- **radius** - A number that sets border-radius of skeleton in `px`.
